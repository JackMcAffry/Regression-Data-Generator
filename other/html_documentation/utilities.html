save_all(params=None, result=None, compression=False, compr_type=None, verbose=False, multi_proc=False, feature_analysis=False)[source]¶
Parameters

  • params – object of class DGP_Params

  • result – object of class DGP_Results

  • compression – bool. Set true if you want compression (must not have compr_type = None)

  • compr_type – str that represent the compression type. Works only if compression is set to True. Can be 3 values:

‘gz’ -> using gunzip compression, file will be saved as <filename>.csv.gz

‘bz’ -> using bzip compression, file will be saved as <filename>.csv.bz

‘xz’ -> using xz compression, file will be saved as <filename>.csv.xz

Parameters

  • verbose – bool, disable if you don’t want verbose output

  • multi_proc – bool, works only if feature_analysis is true. If set to true executes all tasks with multiprocessing (experimental)

  • feature_analysis – if true, will save also analysis on the features
